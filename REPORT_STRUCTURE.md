### 4. Описание вкладок и привязок
Для каждой вкладки указать:

#### 4.1. Вкладка `DefaultBinding`
- Типы привязок, используемых на вкладке
- Примеры свойств ViewModel и их назначение
- Объяснить поведение привязки по умолчанию для `TextBox` (UpdateSourceTrigger по умолчанию).

#### 4.2. Вкладка `TwoWayBinding`
- Примеры двухсторонней привязки (`TextBox`, `CheckBox`, `Slider`)
- Отличия между прямой привязкой к окну и привязкой к ViewModel
- Роль `UpdateSourceTrigger=PropertyChanged`.

#### 4.3. Вкладка `OneTimeBinding`
- Пример одноразовой привязки (например, дата запуска приложения)
- Демонстрация того, что изменение источника не обновляет UI.

#### 4.4. Вкладка `OneWayBinding`
- Пример одностороннего потока данных (источник → UI)
- Описание используемых вычисляемых свойств и/или прогресс-бара.

#### 4.5. Вкладка `Triggers`
- Краткое описание каждого типа триггера:
  - `PropertyTrigger`
  - `DataTrigger`
  - `EventTrigger`
  - `MultiTrigger`
- Описание демонстрационного сценария для каждого триггера.
